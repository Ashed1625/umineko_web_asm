## 🔧 How umineko_web_asm Works

This app uses webAssembly to run onscripter-ru visual novels inside a browser or in a browser-like environment on Windows.

WebAssembly is a fast, low-level code format modern browsers support. It runs closer to the machine hardware than regular web scripts, so the app performs well even on everyday computers.

Here, the app bundles the visual novel engine together with the game content. When you open it, it loads and runs the story seamlessly.

You can expect:

- Smooth graphics and text rendering  
- Fast response to clicks and key presses  
- Easy window resizing  

No installation of extra software or drivers is needed.
